# Washmech

---

## 简介

Washmech是一款**可编程的可以发送和接收特定串口数据的模块**

V1.0版本的模块的主控是使用的是STM32F030F4P6作为主控;有最多可以两个按钮控制串口数据发送,均支持长按短按双击;带有有源蜂鸣器作为响应.V1.0是绝对的低成本方案

---

## 使用方法

每一个版本都是以一个文件夹的形式放出,例如'V1.0'版本文件夹.

每一个文件夹里面都至少包含PCB,Code,3D

其中PCB文件夹是板子的开源文件,可以使用嘉立创专业版打开

Code包含以Vscode+EIDE的开发方式的完整的一个可用工程的Project文件夹,以及专门的主要代码的main_code文件夹;其中main_code文件夹包含必要的库以及main.c,Project文件夹则是完整的工程的zip压缩文件

3D是可用的3D文件,包括SOLIDWORKS输出源文件以及可供拓竹打印机直接打印的文件

**每个版本的功能和方向都不同,详细请见具体版本**

---

## 未来方向

V2或者V1.版本可能会往远程控制以及屏幕输出的方向走,但是随之而来的就是更高的成本,V1.0几乎是最低成本
