# WashMech V1.0

---

## 简介

V1.0是在特定环境下产生的特定版本.它的优势是体积小,成本低,约为3元以下,器件少;供电+5V;有两个按钮位置,可用根据实际情况焊接,建议只使用按钮1即可

## BUG

先说说BUG,由于本人疏忽,忘记对照线序了,所以建议在实际上机之前先自己试一下怎么样才能正确输出

且PCB板上丝印版本是V1.0版本的,则TX和RX的丝印画反了,在后来的V1.1版本修改了,所以如果丝印上是V1.1的就没有反
